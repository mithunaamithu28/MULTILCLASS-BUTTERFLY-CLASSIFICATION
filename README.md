Butterfly Classification using CNN

This project is a multiclass butterfly image classification system using CNN (TensorFlow/Keras) and deployed as a Flask web application. Users can upload an image and get the predicted butterfly species.

📂 Dataset

Dataset used:
https://www.kaggle.com/datasets/gauravduttakiit/butterfly-image-classification

🚀 Features

CNN-based image classification

Upload image & get prediction

Displays uploaded image with result

User login & registration system

🛠️ Tech Stack

Python, Flask

TensorFlow, Keras

HTML, CSS

SQLite
